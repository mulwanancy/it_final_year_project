# it_final_year_project
## project title
Nurseshift management
## project description
Implement a Nurses Shift Payroll System that ensures fair shift distribution and accurate payroll processing based on recorded shift attendance.
To enhance payroll transparency and fairness by automating shift allocation, attendance tracking, and salary computation for nurses working day and night shifts.
